# Nonprofit Grant Tracking and Donor Intelligence Discord Bot

A Discord bot designed for nonprofit organizations to track grant deadlines, log donor interactions, send automated thank-you follow-ups, report on fundraising progress, and alert staff to critical funding opportunities. Replaces spreadsheet chaos with real-time visibility directly in your organization's Discord server.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
